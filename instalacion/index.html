    Instalación y Configuración¶

    Este proyecto requiere la instalación de las siguientes librerías de Python y la configuración adecuada para la conexión a las bases de datos.

    Librerías Necesarias¶

    • sqlalchemy
    • pandas
    • numpy
    • logging

    Configuración del Entorno¶

    Para configurar el entorno, se recomienda usar un entorno virtual. Los pasos para instalar las dependencias son los siguientes:

    pip install sqlalchemy pandas numpy logging

    Importación de Funciones¶

    El proyecto utiliza funciones definidas externamente, todas centralizadas en el archivo Funciones.py. A continuación, se describe la utilidad de este archivo y sus funciones.
